    Booster Pack: Glowing Mushrooms is a small content pack for the early-game Glowing Mushroom biome!

    This mod contains 8 new high-quality weapons (2 per class), a new armor set, and 2 new accessories, all themed around the Glowing Mushroom biome. Booster Pack: Glowing Mushrooms aims to be as balanced and non-intrusive as possible so that it can be used in tandem with larger mods.
